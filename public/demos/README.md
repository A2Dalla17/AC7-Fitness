# Exercise demo videos

Drop one looping `.mp4` per exercise into this folder:

| File | Exercise |
|------|----------|
| `pushup.mp4` | Push-Ups |
| `situp.mp4` | Sit-Ups |
| `squat.mp4` | Squats |
| `jumpingjack.mp4` | Jumping Jacks |
| `lunge.mp4` | Lunges |

Optional poster images: `/public/demos/posters/{exercise}.jpg` (shown while video loads).

Format: MP4 (H.264), muted, loopable, 15–45s, side-on angle for form.

Until videos are added, the gym background poster (`/gym-bg.png`) is used as fallback.
